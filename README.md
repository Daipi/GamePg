## Tai Chi Walking static site

This repo contains a simple static website for a US audience (English copy).

### Files
- `index.html`: page content (intro/about/video/FAQ)
- `styles.css`: responsive styling
- `script.js`: FAQ accordion + footer year

### Preview locally
- Option A (simplest): open `index.html` in your browser.
- Option B: serve as static files using any local web server (e.g., VS Code “Live Server”, nginx, etc.).

### Customize
- Replace the intro/about placeholders in `index.html` (marked as `[... PLACEHOLDER ...]`).
- Replace the video embed URL by editing the `<iframe src="...">` in the “Introduction video” section.
